# Micro:OS
Micro:OS is a lightweight operating system made in PTX for devices such as the Micro:Bit or Pi Zero
# Upcoming Features
- OnScreen Keyboard
- Write App
- Radio Drivers
- Web Chat App
- xCell App
- Pin Header Drivers
- Keyboard Support
- Mouse Support
- Settings App
- ThingAI Instance Code
- ThingAI App
- Image Viewer
- Music Player/Creator
